Profile your GL.iNet router

Capture the device's JSON-RPC API surface — methods, response signatures, and the request shape for writes. Reads only by default; your password is used locally to log in and is never stored or uploaded.

Read-only by default; your password stays on this machine. The profile is the API shape — method names, statuses, and a response signature (field structure + safe example values like numbers and enums), with MACs, serials, secrets, IPs, hostnames and SSIDs replaced by format labels (<mac>, <ipv4>, <string>) — never real values. No device identifiers or raw response bodies. Nothing is uploaded unless you click Submit.
